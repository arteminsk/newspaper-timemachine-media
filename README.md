# newspaper-timemachine-media

Public media repository for Newspaper Time Machine.
Contains thumbnail images only (1600px JPEG).

## URL pattern

```
https://raw.githubusercontent.com/arteminsk/newspaper-timemachine-media/main/issues/[pub]/[decade]/[year]/[mm]/[dd]/thumb_[pub]_[date].jpg
```

## Example

```
https://raw.githubusercontent.com/arteminsk/newspaper-timemachine-media/main/issues/nyt/1940s/1941/07/04/thumb_nyt_1941-07-04.jpg
```
